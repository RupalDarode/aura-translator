## ✨ Features

| Mode | Description |
|------|-------------|
| 🔤 **Text Translator** | Translate any text with tone control + alternatives |
| 📄 **Document Translator** | Upload & translate entire .txt files |
| 💬 **Conversation Mode** | Real-time bilingual conversation translator |
| 📚 **Batch Translator** | Translate multiple sentences at once |
| 🔍 **Word Explorer** | Deep dive — meanings, synonyms, usage examples |

## ⚙️ Setup
```bash
git clone https://github.com/RupalDarode/aura-translator.git
cd aura-translator
pip install -r requirements.txt
streamlit run app.py
```

Add to `.streamlit/secrets.toml`:
```toml
GROQ_API_KEY = "your_key_here"
```
